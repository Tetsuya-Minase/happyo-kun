# プロジェクト概要

## プロジェクト名
**happyo-kun** (発表くん)

## 説明
🎯 Your presentation buddy - Interactive slides with live coding powered by Slidev & Cloudflare

## プロジェクトの目的
happyo-kunは、Slidevを活用したインタラクティブなスライドアプリケーションです。Cloudflareのインフラストラクチャと組み合わせることで、Markdownベースのプレゼンテーションにライブコーディング機能とAPI統合を提供します。

## 主な機能
1. **複数のプレゼンテーション対応**: 独立した複数のスライドデッキをサポート
   - `slides/` 配下の各ディレクトリが個別のスライドデッキになる
   - `/【ディレクトリ名】/【ページ数】` でアクセス (例: `/intro/1`, `/demo/2`)
   - すべてのプレゼンテーションで共有コンポーネントと関数を使用
   - 各プレゼンテーションは独自の `assets/` ディレクトリを持てる

2. **インタラクティブなCSSプレイグラウンド**: リアルタイムCSSエディタコンポーネント

3. **API統合**: バックエンドAPIを呼び出すスライドコンポーネント

4. **サーバーレス関数**: Cloudflare Pages FunctionsによるAPIエンドポイント

5. **レスポンシブデザイン**: モバイルフレンドリーなプレゼンテーションインターフェース

6. **CI/CDパイプライン**: GitHub Actionsによる自動デプロイ

## API エンドポイント
- `GET /api/hello?name={name}` - シンプルな挨拶API
- `POST /api/css` - CSSスニペット処理API

## セキュリティ機能
- セキュリティミドルウェアの実装
- APIエンドポイントの入力サニタイゼーション
- ユーザーフレンドリーなエラーハンドリング
